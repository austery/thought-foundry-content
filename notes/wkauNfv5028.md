### 极低成本云端Agent体验

**杰哥**: 今天**OpenClaw**小龙虾直接满足了大家都有一个土老板的心，土老板都想有一个狗奴才。其实只要大家是做编码的，最近都爆发了，都是一个月挣一年的钱，就是因为那个 Token 量爆了。关于 OpenClaw 的事情，很多时候就是 Token 的费用限制了你的想象力。你省着用 Token 真的就是穷人思维，你一定要往死里用。如果还要省 Token，那就说明你的事情不够重要。要是事情真的重要，你绝对不会想着省 Token。

**小苏**: 没有，那是因为我穷好吗？（笑）

**杰哥**: 你会发现这些新时代的 AI 公司，并不是像过去那种“刘总，哎，你公司多少人”。如果今天还在问这种问题，你就可以去死了。而且最近很火的一个小龙虾对不对？上期聊了小龙虾，大家觉得聊得还不错，问有没有后续更新。所以我最近也自己去装了一个小龙虾。但我觉得我真的没有那个精力去自己部署一台，所以我就买了 **MiniMax** 的云端小龙虾 **MaxClaw**，自己玩了一下，觉得还蛮好的。我现在收回我之前的言论，但是我不收回我的观点，我觉得现在小龙虾对我来说是个刚需。因为我彻底发现，小龙虾这个事情，就是每个人都想成为土老板，而每个土老板都想要有个狗奴才。其实每个人都是黑心老板，只是看压迫谁罢了。

**小苏**: 我懂你了，你是黑心老板，我是土老板，这个没问题，反正大家都想做老板。

**杰哥**: 哎，他那个 MaxClaw 特别好，反正一个月 39 块钱，这是我见过最便宜的了，没有比他更便宜的。这价钱连一只真龙虾都买不来。为什么这么便宜啊？老修，这我搞不懂。

**老修**: MiniMax 现在在全球最大的中间商平台 **OpenRouter** 上用量排第一，就因为它这个价格确实非常便宜，而且智能也够用。在技术上，他们实际上做了一个极致的压缩，激活的参数很少，只有 10B，应该比其他模型小了 70% 甚至 80%。所以这实际上是一个很经济的做法，你实际用起来感觉也够用。我估计你也没做很复杂的事情，对吧？

### 云端与本地部署的博弈

**杰哥**: 确实是够用了，而且省去部署的折腾，我整个心情就好多了。之前你部署在自己电脑上的小龙虾，我们玩得很痛苦，网关要自己搞，容易搞崩溃。其实各家也都有自己的小龙虾，**Kimi** 也有，你之前部署的那部是 **GLM** 的对吧？

**老修**: 对，GLM。因为我们是它的编程客户，所以正好有送。但是送的东西就是那样，用着不是很爽，关键时刻一抢流量就不够用了。相比之下，MiniMax 速度够快，所以你不会觉得太卡。用其他家的模型，在上班高峰期能明显感觉到反应变慢，回复的质量也比较差。Kimi 我没用过，应该比 MiniMax 贵多了，大概要大几十或者一百多一个月。但它质量还比较不错。我觉得在云端跑 Agent 我绝对双手赞成，因为整个经济性、稳定性和安全性肯定是最好的。

**杰哥**: 但是对比我们本地跑的，云端的就跟瞎子一样。因为你实际上没有办法远程连一个桌面到云端，授权浏览器给它用，你等于是在抓瞎。除非你明确告诉它要安装哪些 skill，否则第一次用的人会很痛苦，有种盲人摸象的感觉。这是我自己的体感。

**老修**: 可能还是你现在还不太会用。其实小龙虾的用法，还是要尽量让它去调用各种成熟的工具。比如编码，让它用 **Claude Code** 或者国内的 Cursor。但这需要配置，你跟小龙虾聊也是要配的。所以为什么现在国内有人收 499 帮人配置，甚至一两千块钱到企业里面去起步配置，这些服务还是有价值的。这就跟以前企业里面的 IT 帮忙装电脑一样，你得给这个小龙虾配好相关的软硬件环境和网络，要是全靠自己弄就会很累。

**杰哥**: 这个跟装电脑还是不太一样。你可能要预制很多默认的 skill，或者要设计好这个 Token 要怎么消耗。我现在了解了，这东西就是要以消耗 Token 来换智能。

**老修**: 是的。我用装电脑比喻只是觉得比较好理解，反正装什么东西便宜、快，就可以直接上。

**杰哥**: 至少从我的选择来看，我肯定选套餐便宜的，背后就是它的 Token 便宜。我后面有一个领悟，就是用这东西不能省 Token。像我们以前用那“预三家”会省着 Token 做路由，用 Claude 做架构，现在用 **Cursor** 来写代码、抓数据写前端，安排得明明白白，在舒适区里面自由奔跑。

### 告别聊天：任务驱动的AI

**杰哥**: 我的感觉跟你们不一样。节前大家聊小龙虾，后来你们过节去了，我在这边没节可过，就开始装小龙虾、折腾它，结果搞得好累。以前用 AI，可能更多是在 Gemini 或者 OpenAI 里面对话，突然一下上了强度。上了小龙虾之后，你希望它不光是换个方式跟 AI 对话，而是要真正去干点事。就跟小苏说的，每个人都想当老板。但当老板不是坐着喝茶聊天，你得真得给手下安排事情，得让 Agent 干活才行。去给它找活干，在这个过程中你会体会到，当老板其实不容易，难在让手下人尽其用。
我现在虽然还没真正体会到云端小龙虾的高级用法，但在本地部署的过程中学到了很多东西，包括要了解一些简单的数据库和编程基础，才能知道自己做的一步是对是错。所以在这个过程中，其实不是我养龙虾，而是龙虾在养我。

**小苏**: 最近小龙虾已经成为 GitHub Star 榜第一名的项目，这让我非常震撼，它甚至超过了 React 框架。

**老修**: 超过 React？

**小苏**: 对，超过 React。我觉得这是 AI 行业非常里程碑的一件事情。去年的 Minds，今年的小龙虾，而小龙虾的势能比 Minds 大很多。它最重要的点在于，能够让每一个哪怕不懂 AI 的人也想搞只龙虾来玩，让它干活。这是 Minds 当时做不到的，Minds 还是圈内人感兴趣，有营销成分在里面。但 OpenClaw 这块，Peter 不可能去投流，它是自发破圈的。

**老修**: 刚才小苏也说了，MiniMax Token 在 OpenRouter 上已经排到第一了，也就是中国模型里面排第一。我觉得这都是 OpenClaw 这件事的衍生。而我是觉得，真的是开始用这个小龙虾之后，才开始在乎 Token 使用量这个事。我后面也发现，省着用 Token 真的就是穷人思维，一定要往死里用。之前有一段时间我真的很节省着用 Token，尽量先把编码做好。

**杰哥**: 那就说明你做的事情不够重要。要是事情重要的话，你绝对不会想着省 Token。

**老修**: 没有，那是因为我穷好吗？（笑）穷人思维，我先批判我自己。但我是同意的，在小龙虾出来以后这个节点，是不能省着用 Token 的，因为很多时候就是 Token 的费用限制了你的想象力。

**杰哥**: 对，比如用小龙虾，我会想我有一套成熟的工作流，要找 3 个助理组团拉个群做某件事，我把每个助理的工作流排好，固定的工作写好程序让他们去执行，这样肯定是最省 Token 的。但实际上，这样的操作会让我错失很多智能的机会。虽然用了 AI，也比别人会用一点，但获得的智能并不一定比别人多，因为这没有超过我的想象力。

**小苏**: 说句不好听的，你可能还是在做高级版的自动化，而不是 AI。

**杰哥**: 对，还是走老路子去装那个旧系统。这确实是我最薄弱的地方，我是犯过这个错误的。而且真的是用小龙虾之后才开始在乎 Token 消耗。以前是买个 ChatGPT Plus 或者会员，现在要考虑从 API 上到底花多少钱，这些 Token 到底干什么用，智能能不能换成钱。

**小苏**: 真没算过。这就跟算你的员工一样，带薪拉屎也是带薪，你还得给他买纸，对不对？

### AI公司财报与非线性增长

**老修**: 所以这个闲哥，我觉得特别有意思。我想听一下你从财务的角度怎么看。这次 MiniMax 在电话会议上算了一笔账，CEO **严俊杰** 说，他们最新的模型 **M2.5** 以每秒 100 个 Token 连续跑一个小时，成本大概是 1 美元。也就是说，1 万美元足够让四个 Agent 跑一整年。我没仔细算过这笔账，被他这么一算，感觉还挺便宜的。我们做编码一秒钟跑 40 到 60 个 Token，100 个 Token 已经是日常两倍的速度了。如果按日常使用，可能只要 5,000 美金就能跑一年。折合人民币也就 3 万多块钱，一个月 3,000 多。现在招个实习生都不止 3,000 块，何况这是四个 Agent 给你干活。换算成人力，你会不会觉得特别便宜？

**闲哥**: 我觉得对于 AI 公司，千万别这么算账。这是我的一个暴论。

**老修**: 展开讲讲。

**闲哥**: 我确实看了它的财报，因为我关注港股。它是 AI 模型公司里边第一个发财报的，而且还开了电话会议。你看他这财报是截止在 2025 年 12 月 31 号，而它是在 2026 年 1 月 9 号上市的。这中间其实没有反映出上市之后的表现。现在已经 3 月份了，2026 年都过了两个月，而这两个月可是翻天覆地。它上市的时候还没有 OpenClaw，也没有现在这么频繁的调用。
所以我觉得，刚才说 1 万美金让四个 Agent 跑一年：第一，Token 的成本以后还会再降。随着模型推理能力和电力等各方面的提升，Token 成本肯定会降，所以你现在算的数和 6 个月以后看可能完全不是一个量级。

**老修**: 穷人思维，对不对？

**闲哥**: 对，所以你不能线性地看 AI 公司。它财报里面有个数据特别逗，它整个 2 月份的 **ARR**（年度经常性收入）已经达到了 1.5 亿，超过了它 2025 年整年度的收入。这怎么比？你不能说它 2025 年没干事，人家干了好多事；那你人说它 2026 年 2 月份干了特别多营销吗？也没有，我看它营销费还没怎么花。
我觉得 2 月份有点魔幻，不止 MiniMax，Kimi 2.5 发布以后，整个 2 月份发了 20 天就超过去年全年的业绩。不要说 Kimi，Claude 这两个月也赚了很多钱。所以不要用线性的思维去考虑 AI 公司的财务数据和表现。
以我用小龙虾的个人体验（纯主观，没有任何数据基础）：首先，当你开始用 AI Agent 和你一起工作的时候，你的工作方式就改变了。你不能再按 24 小时来算。我今天精力充沛工作 10 个小时就到头了，不能连续一年都这样工作，不然就挂了。但是 Agent 可以，而且你搭了一个 Agent 之后，就可以有第二、第三、第四个，它们可以一起干。这和碳基生物完全不一样，没有 996、没有 007 的概念。

**老修**: 很多年前有一本书讲人月的概念，里面有个例子特别对，就是《**人月神话**》。

**闲哥**: 对，你不能找 9 个女人一个月就把孩子生出来。意思就是说，你找更多的人出来，沟通成本很高，同时干一件事不见得干得更好。这跟扫地一样，你即使有 9 把笤帚，也不见得比一两个人扫地更快，因为空间有限制。但是用 AI Agent 在编码、debug，公司的工作效率、产出和研发能力和以前是完全不同的。而且 AI 本身还没有到中场，还有很大发展空间。这样看来，未来一个月就能干以前一年的活。所以我说不能线性看这事，现在纠结 5,000 块还是 2,000 块没有意义。更重要的是赶快把 Agent 用起来，从 1 个到 3 个、5 个，每天你睡觉它干活，那你离每天喝咖啡就不远了。

### 2026年AI变现的三个方向

**老修**: 只要是和编码挂钩的，最近都在爆发，都是一个月挣一年的钱，因为 Token 量爆了。而且海外也是这样。我们一直聊的 Cursor，最近三个月它的 ARR 直接翻倍了，从 10 亿美金直接翻到了 20 亿美金。

**杰哥**: 他不是都快挂了吗？怎么突然翻倍了？

**老修**: 并没有。我特别去看了一下 Cursor，它最近上了一个头条，ARR 直接达到了 20 亿美金，是三个月内翻倍的。另外 Claude Code 增长也很快，到 1 月份又翻倍了一次，现在达到 25 亿美金以上。短短一个多月，大家都多赚了这么多，真是天降横财。

**闲哥**: 这种情况之下，大家的预判会不会全部都飘了？都觉得这钱太好挣了。

**杰哥**: 我觉得是因为大家和 AI 对话的模式变了。不再局限于聊天对话框或者做虚拟女友，而是真的开始完成任务。找到了命令行方式，或者通过 OpenClaw 直接用语音让它管理和调用现成模型与工具。这个过程中的 Token 消耗是基于任务的，不再是基于情感的。

**老修**: 没错。

**杰哥**: 这不是一个指数级的消耗。如果基于聊天，消耗还算得出来；但如果是基于任务完成，大家对 AI 的期待就变了。今天大家看到聊天框的产品都已经快吐了，不要再给我聊天框，直接像 Minds 那样给我一台电脑，帮我把任务完成了。今天小龙虾直接装在本地电脑上，帮你把牛马的工作给做了，满足了大家想当老板的心。我觉得这是好事，人类很难用理性去撬动，往往是用情绪撬动。如果天天像 GPT、Claude 这样按部就班地升级，很难激发大家。小龙虾卡在了一个很好的情绪点，带来了整个 Token 消耗的大爆发。

**老修**: 这里面最重要的是，现在大家觉得它只是个玩具，因为还没用它挣到钱。一旦哪天它能挣钱了，那完全不一样。

**杰哥**: 对，你现在不熟所以不敢让它干活，真的熟了，发现它在某些地方的智能甚至超过你，你肯定会拿它来赚钱。

**小苏**: 今年看下来，有几个方向是很确定能赚钱的。第一个是**视频生成模型**。视频生成是 Token 消耗的大户，也是 MiniMax 想去抢的市场，包括 Google 也在抢。这一块是能直接赚钱的，现在那些做漫剧的公司，把 AI 接进来开始流水线生产。我最近很喜欢看那些 AI 生成的漫剧，比如“带你穿越体验清朝太监的一天”，确实挺好看的。很多做视频生成和漫剧的朋友在这一块能直接算出 ROI，去投流然后转化，商业模式已经很成熟了，用 AI 核心是把成本砍掉，就能赚钱。
第二个是办公市场。AI 办公今年比 Coding 更有潜力，**Co-work** 出来以后，虽然讲的人不多，但今年会撕开 AI 办公市场很大的口子。这一块消耗 Token 也是个大户。员工偷偷用它摸鱼，做出来的效果还好，老板照样发工资。
第三个就是编程，编程也是 Token 消耗大户。老修最有发言权。

**老修**: 我们做 Coding 哪怕不赚钱也必须得买，每家都有用，GLM、Kimi 还有 MiniMax 都在用。每一家的品味稍微有点不一样，我是三家都买了。

**小苏**: 所以今年这三个方向下来，哪个不是 Token 消耗大户？谁不把 Token 成本降下来，谁就是没有功德。

### 混合模型的高性价比策略

**闲哥**: 在这个时代，到底什么在未来更值钱？互联网时代是流量最值钱，所以字节和阿里掌握了流量就比较牛。但现在，最重要的是掌握 Token 的性价比——也就是输入输出的智能与消耗 Token 之间的产出要合适，同时花的钱要合理。
现在海外用 Agent，大家都不局限于某一家的大模型。比如用 Claude 来分解任务，用 ChatGPT 做调研，用 Gemini 做动态输出，中间执行层用这些东西，最后用 MiniMax 这种便宜的中国模型做输出。整个任务算下来，80% 都是便宜的执行层 Token，高级 Token 只占 20%。这样性价比极高，又能有产出，甚至能靠它赚钱。这会是未来非常典型的 Agent 使用方式。在这点上，我很看好国内高性价比 Token 输出的能力。

**杰哥**: 所以新时代的 AI 公司不再是过去那种大家握手问“你公司多少人”，问这种问题的人可以去死了。

**小苏**: 别有用心（笑）。刘总说我公司就三个人，那今天肯定很高兴。或者说“苏总咱们去哪打球，事情都让 Agent 干了”。有什么事把秘书的电脑拔回去，有几个 skill 复制在 GitLab 上，自己去下载就行。

**杰哥**: 没错，所以未来很多老板不要一见面就问你公司有多少人，应该为此感到羞耻。

**小苏**: 放心，物以类聚，人以群分。以后的问法是：我公司有多少人，有多少个 Agent。

**闲哥**: 今年是一个分水岭，负责智能的 AI 和负责智能以下的苦力 AI 走向了不同的路。2024 年和 2025 年中美可能还在工程与智能融合，但今年，负责苦力这一块的中国优势开始显现出来了。

### 中国把AI干成制造业
