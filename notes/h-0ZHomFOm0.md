大家好，欢迎来到Money or Life美股频道。我是Ace，今天是大年初一。Ace 祝屏幕前的各位新春快乐！希望大家在新的一年，一马平川，马到成功！

本期视频，我把手机架在电脑前，用带回家的麦克风和大家闲聊，几乎没有剪辑。聊聊我作为一名 YouTuber，如何善用马斯克现在的 X 平台进行辅助美股投资。当然，除了 X 平台，我还想聊聊我自己，以及作为一些会员朋友们心中“意见领袖”的角色，是如何去使用 X 平台，以及我的价值定位是什么样的。

### X平台：美股投资信息新战场

我想，过了这么多年，大家依然对当初 GME 这家公司，散户大战华尔街的事情津津乐道。从那时开始，似乎机构和散户变得比较对立。这当然不是事实，其实大部分投资、大部分公司的波动还是机构在主导，但不得不说，散户的声音越来越重要。“信息平权”这个词，在美股投资上体现得非常明显。

会员朋友们应该知道，我现在几乎每天都会将我在 X 平台上看到的关于公司和产业的新闻分享给大家。主要分享在三个平台：第一是微信群（我的会员群）；第二个是推创 APP，它有内置的聊天功能，我也可以发图片；第三是 Discord，里面有一部分朋友加入了 Discord 聊天群，这个级别我也会分享在 Discord 聊天群。

我之所以乐此不疲地分享，是因为我觉得现在 X 平台，特别是马斯克收购并更名为 X 之后，这个平台能给美国投资者带来的有用信息，远远大于有害信息。我自己也自认为有能力去分辨哪些是噪音、哪些是有用的信息。

### 信息甄别：从“噪音”到“价值”

提到噪音，有一种说法是：如果你是长期投资者，要忽略很多新闻，因为这些新闻都是噪音。但是现在，我觉得这个说法可能有待商榷，或者有些不对了。你想，五年前、十年前，你要得到公司的消息是不是很不容易？除非你是行业内的基金经理或者一些大佬，可能才能得到一些内部消息。但现在，真正实现了信息平权。

在 X 平台，不仅有公司的官方账号，还有公司的 CEO 以及各大高管。这些 CEO 和高管，他们既是公司的领导者，也是作为个体在 X 平台上发声。还有像马斯克这样的超级大 V，他说的每一句话，对于他领导的公司甚至整个行业都有引导作用。

所以，我们每天能在 X 平台上发现的信息很多。我自己也花很多时间在 X 平台上不断刷消息，以至于我现在手机上的其他社交网络，比如小红书、Meta（旗下应用），我几乎都不使用了，可能几天才点开一次，除非有重要的消息推送给我。我大部分时间，因为我现在做美股投资相关的视频，所以一有空就会看看 X 平台上我关注的公司和行业的重要消息，然后马上分享给大家。

我认为，现在对于屏幕前的大部分朋友来说，投资美股的门槛是比较低的。但请注意，我说的门槛比较低，并不是说更容易赚钱，而是说如果你肯花时间花心思，其实你有很多渠道去了解公司。相较于以前，你可以更容易地了解到你感兴趣、你想投资的一家公司。

### 博主价值：信息筛选与整合

除了刚才提到的 X 平台上公开的、面向所有人的消息之外，也有一些所谓的网红或博主，无论是英文区还是中文区都有。以我自己为例，我的定位是：我并非比屏幕前的你们有更长的投资经验，也并非金融专业出身，甚至都不是功课出身。但是我有时间。我去年开始全职做博主，所以有很多时间帮大家分析公司的官方 **SEC 文件**，分析公司官方的所有消息。

其次，我每天可以看很多 X 平台上的消息。我就通过自己的方式，比如用自己的表格、自己的视频，把这些消息“去其糟粕，取其精华”，去掉那些假的、我觉得可能不对的，把有用的集中起来。无论是通过个股分析、周末视频，还是每日新闻分享，都整理后分享给会员社群。

以上分享主要是想告诉大家，如果有条件的话，尽量去 X 平台上关注你所投资公司的第一手消息。这些消息可以来自公司官方账号，可以来自公司高层，当然也可以来自一些大 V。为什么？因为很多大 V 现在了解消息很快。比如 **Tesla**，他们有几个在 X 平台上的所谓“非官方代言人”。这几个人虽然不是 Tesla 员工，也不是 Tesla 官方挑选的，但他们说的东西基本上都是真的。

如果真的没有时间或条件，可以考虑关注一些博主：首先，要和你世界观、三观相符；其次，和你的投资观比较接近。这些博主类似于我自己，我们花了时间为大家总结了公司的消息，呈现给你，这样你不用花这么多时间。其次，如果你觉得这些博主的观点（投资观点这种主观观点）对你有用，也可以作为一个参考。毕竟我们每个人时间有限，不可能花特别多的时间去关注特别多的博主。比如我自己现在全职做博主，以前我会付费关注一两个博主，看看他们在说什么，但现在也没有时间看了。
