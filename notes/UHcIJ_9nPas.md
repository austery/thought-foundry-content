### 老龄化背景下的退休返聘潮

面对劳动力老龄化，国家机关开始返聘退休人员。这个是挺有意思的，因为现在其实失业率很高，然后国家机关又是香饽饽，大家都愿意去做。但国家机关如果招收一个新的人，你要负责他一辈子的社保很麻烦，所以国家机关开始搞这个返聘。

最近一个例子是四川某大学的电子工程学院教授。他是达到六十二岁的退休年龄之后，今年九月份以研究员身份重返教职。返聘的好处是他的薪水比退休前低，而且学校就不会去缴纳社保了。所以我觉得中国未来会有返聘潮。返聘有两大好处：第一，退休人员是不用再缴社保的，他的社保缴纳已经结束了，所以聘退休人员是节省一大笔开支。第二，退休人员工资可以比以前低。对于企业来讲，想来想去可能会比招一个新的年轻人要容易。

那这岂不是进一步压缩那些年轻人的工作岗位了？因为我看中国博士的就业非常难，好像每年有十万博士左右要进入就业市场。因为进入大学已经是极为艰难的了。

最近其实还有一个就业的难点，就是社保缴纳的刚性非常非常强。因为之前有一个非常重要的判例，如果员工跟用工方达成协议说你不用交我的社保，你把钱折给我。如果员工主动说我就是不要社保，法律这是强制性规定的。之前浙江的判例是支持，如果员工方说我就是不要社保，那最后如果员工方来告的话，企业要补缴，但员工方需要把这个钱全部退还给企业。但是北京地区的判例就更强化用工方的责任。现在全国在向北京的判例靠拢，导致企业的压力更大。反正最近越来越强，可能就会导致很多地方愿意去聘一个返聘不交社保的退休人员。

在日本也是这个道理。为什么大家愿意雇佣打工的人，因为不需要交社保。基本上雇佣打工的人的成本是交社保人的二分之一左右。在国内因为交社保的基数实在是太高了。

大家也肯定很关注最近中国模型，就是 Kimi K3 和阿里通义千问。我本人对于模型有两个看法。第一，我是依然相信 Scaling Law，就是模型参数越大，效果越好。Kimi K3 和阿里通义千问的参数都是 2.8T。参数会有个小问题，阿里的通义千问 3.8 Max 是开源的，可以本地部署，但到 2.8T 基本就没有什么本地部署的可能了，你都得交给他来部署。第二，我也相信运算中的注意力机制和激活参数的能力。中国模型之所以便宜，就算它的总体参数比较大，它便宜的原因就是它激活的参数比较少，它的注意力比较稀疏。到底 Kimi K3 的注意力机制是什么样，也是技术面讨论的一个重点。AI 这个事就是一分钱一分货，参数越大，基本上效果就是越好。而且在参数越大的基础之上，每一次激活的参数越多，注意力越集中，以至于到 Dense 模型，它的效果就会越好。我觉得这方面没有什么可质疑的。

中国模型在某些任务上，尤其是 Agentic 编程任务上的效果确实比较好。因为这些任务第一比较容易蒸馏。第二也比较容易有标准答案来训练这个模型，它比较容易程序性验证。比如说 Kimi K3 在长程 Agentic 编程上应该能排到第三，仅次于 5.6 和 Fable。但是 Kimi 在知识的准确性上就要差一些。看 Omniscience 那个模型的分析，可能就要排到第五或者第六。现在 Kimi 的总分是排到第八，是落后于 Claude 3.5、Grok 4.5、Opus 4.8、Gemini 3.1 Pro、Fable 5、5.6 和 5.5。

比如说模型的精确性，它相对比较差，精确性只有四十六，排到第八名。幻觉率 Kimi K3 其实排到中段了，百分之五十一的幻觉率。泛化能力评分相对来讲比较差，排到第八名。所以我们不能只看他在排名上比较好的部分。Kimi K3 相对比较强的是医学知识、科学工程学的知识。这些知识强其实都是相对程序验证能力比较好的，软件工程能力比较强。Kimi K3 比较弱的就是法律知识和商业知识相对弱一点。整个中国的模型比较强就会强在工程学、科学这些有程序性验证答案的，在其他能力上相对比较弱。在商业知识上能力还不错。

Kimi K3 还有一个小问题，就是 Token usage 非常非常高。就是 Kimi K3 是一个很吃 Token 的模型，即便它的成本可能是 Fable 5 的百分之六十。但如果你最后实际完成一个任务，有可能最后花的钱要多一点点。现在 Token efficiency 优化比较好的应该是 OpenAI 的 5.6。

我的观点是，中国模型在 Agentic coding 上肯定发展是可圈可点。包括 DeepSeek、智谱、Kimi 都不错。如果你评价他们，是非常好的 Base model。Kimi 的 Base model 看上去确实比 DeepSeek 和智谱的 Base model 能力要强，本来参数也要大的多。所以我不完全否定中国模型的先进能力。但是我认为它是一个第七第八名的水平。如果综合来看的话，它在某些 Agentic coding 上可能是第三名。认为它已经是锁定的第三名或者第一名，我觉得是无稽之谈了。甚至 Kimi 官方都没有说自己是第一名，大概说自己是第三名，落后于 Fable 5 和 OpenAI 5.6。

我看外媒对这个软件的发布非常重视，评价都很高，认为缩短了中国和美国 AI 方面的差距。这个我不好说，因为现在 AI 是个田忌赛马的时代。就是你拿你的上马去对别人下马，然后你拿你的下马去对别人上马。

### 警方搜查手机密码的法律分析及应对

最近国内有一个文章在流行，关于有一个人被抓了，警察问必须交出手机密码。很多国内外的朋友可能都不是特别理解。我简单说一下警察问的这个笔录，这是一个办案律师公布出来的：

警察问：你为什么拒绝提供手机密码？
嫌疑人说：里面有我的个人隐私，我不想让你们知道。
警察说：我只是看和案件有关的转账记录和信息，不会泄露你的隐私。
嫌疑人说：我不信，我的支付宝和微信还有钱呢，万一被你们知道了密码，把我的钱转走怎么办？
警察说：我们和你签署了协议，上面有清晰的时间。如果在这之后你的款项转出去，就由公安机关负责。这个你要放心，你能不能配合？
嫌疑人说：我信不过你们，你们不要再问我了。

警察思想教育了十分钟之后问：现在可以提供手机密码了吗？
答：可以。

有很多评论，有专业人士的评论，也有专门写文章的评论。我快速说一下这个结论：在中国，通信是受宪法保护的。我的结论就是任何警察没有任何理由来强制检查你的手机通信。如果你拿搜索票，我认为也是不行的。因为我们的法律规定是可以对你的住址、人身进行检查。这里面有一个问题，我来搜查你电脑当中内容，我让你打开密码，法律并没有强行让你说出你的密码来，我说忘了就忘了。这里面一个重要的法律原则就是任何人不能自证其罪。我没有义务配合你打开我的电脑，我说忘了就忘了。警察来搜查你的手机、你的电脑跟你要密码，其实是没有法律依据的。

但警察有一个办法，比如白纸之后好多人被警察传唤去，都要解锁手机。如果不解锁，警察想必拘的这一段时间，他的行政裁量权是有了。你要给了可能六个小时就走，你要拒绝解锁，他可能就拘留你七天。这需要根据你自己那个案子来权衡。如果是比较严重的案子，交出密码只会加重对自己的定罪。

另外，在国内很多情况下，你的密码并不是国保、公安破解的，他是追踪你在机场、火车站使用电脑和手机密码时，从摄像头看到你是怎么输入密码的。所以大家要普及指纹解锁，或者输入密码时要盖住。如果在国内的公共场合，警察直接扑上来给你打晕了，拿你的指纹解锁就完了。

国内一个当事人告诉我两个经验。第一，警察踹门进来的时候，第一件事就要合电脑。只要有人进来一定要合电脑，因为警察扑上来首先是控制你的电脑，你合不上他就可以做很多工作。第二，如果感觉说出来会更增加自己的危险，一个非常著名的律师教的，就胡乱说一串密码，其实说假的，说两次手机就封住了。

如果大家有需要的，一般手机都有强制 reset 按键，不会很复杂。比如警察进来时，手揣到兜里就能够操作强制 reset。大家一定要备一下自己手机的强制 reset 方法，如果你真的有这么严重的话，你把这个背下来，可能就三四秒你的手机就清空了。警察掏出来就是一部砖。

所谓续命就是说它的产业规模占GDP的比重是多少，它的就业能力占总就业比重是多少，它的财政能力占总财政比重多少。这几个行业强不重要，关键是它对于整个经济结构性的影响多大。它的产业规模大概是GDP的10%，然后就业规模只能到1%。它整个就业存量基本是外卖送餐的一半，只有八百万人就业。财政续命指数是负数，为什么是负数呢？因为出口退税和补贴实际上就是好看不中用。实际上国家得给它贴钱，按照我这个算法一年贴619亿。而且因为地方政府的补贴是不公开的，所以实际上这是一个很保守的值，国家补的更多，也就是说不是一个市场性竞争。欧美或者德国操心这个是另外一回事。政府就只管它续命，管它对于中国经济的贡献多少。

那么我这个东西靠不靠谱呢？我认为在现在情况之下算非常靠谱的了。简单讲讲构成，现在最容易查到的可信的公开数据是两部分。第一部分是出口数据，出口数据的营业额相对可信。第二部分是上市公司的数据相对可信。我找到了几乎这些领域所有的上市公司，上市公司的营业额覆盖率在50%以上。通过上市公司的数据，能看到行业的利润率，能看到每年出口退税的额度和政府补贴的基本额度，也能看到每个企业的招聘人数。全部数据有统计学方法在里面，结合非常多各个行业口径的行业营业额来看中值是多少，也看方差。只要方差合理才能用中值减去比较合理的出口量，就能看到内需的指数。现在在国内最不公开的也最难的就是行业总的营业额数。这个总营业额只能通过三四个不同的指标反算，但因为有这几个值互相印证和上市公司的财报做印证，所以最后的结果是比较可信的。

### 五大新兴产业续命指数的测算过程与价值

但这个是要通过非常复杂的运算，就算出这些数，大概起码也是得有十二万行的数据来算。当然这个是必须跟 AI 协作才能够处理这么大部分的数据。所以说这是一个非常非常复杂的一个计算，我也花了很长的时间来算出来。所以这里面最有意思的就是这个五大新产业续命指数这个值有意义，CPI、PPI 有益、社融有益、社零这几个就比较简单一点了。这几个就是从社会融资、社会零售和 CPI、PPI 结构源把扰动因素减去来看那个最终值。

然后有意社零里面比较有意思的是，那个里面有一个线上的单位社零和线上餐饮同比。那两个能看出中国的消费降级的状况是非常非常直观和直白的。但这里面我觉得最有技术含量的就是这个续命指数。因为续命指数本身也可能大家比较关心吧，就到底它能有多少。那这里面呢就是通过这几个比较靠谱的公开数据，再加上很多其他数据综合性交叉运算来得出来的。它从经济学上并没有经济学理论上的创建，这个所谓的续命本身的道理也非常浅显。但它的关键技术性就在于如何在中国这么一个高质量公开数据非常非常少的情况之下，怎么通过综合数据把它算出来，算到一个比较靠谱的区间上来。

### 续命指数对中国经济现状的反映

虽然不懂这么比较深奥的，但是我感觉那个百分之一的数据，大体上符合我自己对中国经济的那种观察。因为比如说我们山东那一带的人，在有外贸的企业里工作的人极少。但这还不是总体外贸，这只是这五个产业。因为其他的比如说制造业外贸或者家居外贸，纺织品外贸不算。

也就是说中国那么大的经济总量，单靠外贸是根本撑不起来的，不具积体性的。外贸内贸靠这五个产业都支撑不起来了，就这五个产业已经是最火的，都支撑不起来，基本上就是跟踪。当然这个跟踪呢，一是看现在的这个静态值。第二个呢是看趋势。

比如说都是从二零二零年之后算到现在，从二零二零年到现在，其实产业规模变化比较有限。因为从二零二零年初这几个产业五大产业占 GDP 就是百分之八点一二了。这六年期间也增长了不到百分之二，所以其实增速没有那么迅猛。然后呢，整个就业能力当然是随着这个产业的升级，慢慢在上涨的。但是净财政能力是在下降的。进财政能力在进入到二零二四年国家大规模的刺激这些产业之后，还有比较大的下降。尤其是里面集成电路板块增长太快，但集成电路板块是国家补贴和退税退得最厉害的领域。

所以其实这个政策扶持的结果，对其实整个净财政贡献能力呢还是不断在下行的。所以基本这么一个情况，也就是它的上行速度非常有限，但财政下行速度特别快，然后就业的上涨明显慢于产业规模的上涨。产业规模上涨也不是特别显著。涨了六年，涨了不到两个点在 GDP 里面。而且最近这个内需下降速度还在加大，但是外需补得挺厉害的。大概是这么个情况。

### 续命指数模型的更新与结论

这个数据模型每个月都会根据新的数据不断往上加，每个月会更新一次，形成一个持续的发布。这个也是跟李老师他们合作，会在他们那边发布。因为影响比较大，所以这个经济续命指数每个月都会在他们的平台发布。除了这个之外，每月应该还会发布其他几个中国宏观经济分析的报告。

关于这个指数为共产党算命呢，是我们大家最愿意做的工作。这个续命指数最终结论是什么？我只是给一个参考，我的结论很简单，就是续不上。因为中国现在很大的问题是财政问题和就业问题嘛。起码结论就是这五大新兴产业对这两个问题的帮助非常有限，甚至财政的帮助还是负的，就是这个结论。
