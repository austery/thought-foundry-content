### 年度最佳产品提名：Claude Code、Gemini 3 与 Nano Banana

**小苏**: 大家好，今天进入到我们的人民公园说 AI 年终颁奖。虽然我们只做了一年，但其实是“一年两度”，因为 AI 圈一天，人间一年，上半年我们颁了一次奖，现在进行年度颁奖非常有必要。我们这次会颁出 5 个大奖。首先是第一个：**年度最佳产品奖**。这是几个人心中最重要的奖项。老修，你心中的 2025 年度最佳产品是什么？

**老修**: 作为做产品设计和工程实现的人，我的提名没有悬念，就是 **Claude Code**。它今年 3 月份左右刚出生，现在已经住到你的电脑里，成为了专业的工作搭子。我今年有七八个月的时间都在上面写代码、搞原型、搞测试、搞运营，甚至自动化数据处理。虽然它叫 Claude Code，但我看它可能想改名叫 **Claude Agent**。它不再是远程的云端对话框，而是真正通用的 Agent。最新版的 Claude Code 已经支持原生控制 **Chrome 浏览器**，带着登录好的 Session 帮你完成任务。它就像早期的特斯拉跑车，虽然现在门槛稍高，但性能极其优越。它今年为公司贡献了超过 20%-30% 的营收，已经服务了 30 万家企业。

**小苏**: 我能理解。Claude Code 开启了本机 Agent 时代。我虽然不会技术，但我用的是 Google 的 **Antigravity**，它不仅能写代码，还能帮我管理本地文档和协作。老修，上半年你颁给的是 **Cursor**，现在为什么“不香”了？

**老修**: Cursor 发展得也不错，它把技术向的东西提炼成普通人能用的 GUI，破圈做得最好。但我更看重生产力的提升和稳定性。Claude Code 让企业真刀真枪地买单，收入从 2023 年不到 1000 家客户跑到了现在的 30 万家。Cursor 更多是个人用户，而 Claude Code 带来的企业级生产力提升更稳健。

**小苏**: 明白。Cursor 停留在应用层面，没有跃迁成全能 Agent。这需要大模型底层加 Agent 架构。杰哥，你的提名是谁？

**闲哥**: 我比老修长情多了。上半年我颁给 **Gemini**，现在年度颁奖我还是要颁给 **Google** 的 **Gemini 3**。尤其是它出来后，我对它的信任度上升了一个档次，把它当成良师益友。以前是一个问题扔给三四个 AI 对比，现在我会首先选择相信它。无论是图形生成、图表还是 UI 交互，它都能玩出很多花样。举个例子，我最近在温哥华弄餐馆，需要买二手设备。我把设计方案扔给它，它帮我在二手市场比价，甚至跨国帮我找到了美国的一个二手排风机。它给出的建议非常智能：让我把两个产品都买回来，留一个卖一个，把路费和产品钱都赚回来。这种智能已经达到了可以平等交流的地步。

**小苏**: 你之前是 **OpenAI** 的 Pro 用户，现在呢？

**闲哥**: 我还是会用 **GPT**。Gemini 严谨流畅，而 GPT 或 **Grok** 有不同的风格。面对复杂问题时，我会用不同的 AI 来实践批判性思维。虽然阅读量增加了，但学习能力变强了。Gemini 3 确实帮我省了钱，这知识付费太值了。

**小苏**: 轮到我的提名了，我提名 **Nano Banana**。理由有三：第一，它今年养活了很多人，诞生了无数套壳的生图、PPT 辅助产品；第二，它的设计极其极简，就像 Google 搜索框一样，交互只有提示词，但功能极其强大，指哪改哪；第三，它集结了 Google 多年的商业模式和数据积累，它理解物理世界、识别定位、知道光影，它不是 2D 的，而是 3D 的。

**老修**: 确实，安德烈·卡帕西（**Andrej Karpathy**）也说 Nano Banana 开启了新范式。它可以理解黑板上的数学演算推导，还能根据一张风景照准确猜出地理位置。它直接改变了“抓小三”行业的规则（笑），因为它能识别几点几分的飞机轨迹和光线角度。它对世界的理解层次已经超出了我们的想象。

### 最终结果：Nano Banana 开启 3D 世界理解新范式

**小苏**: 经过一致决定，年度最佳产品奖颁给——**Nano Banana**！它开启了下一个时代的拼图，在端侧能力和世界模型上都有巨大突破。

### 年度最佳团队：OpenAI 的极速进化与谷歌、阿里的生态整合

**小苏**: 第二个奖项是 **年度最佳团队奖**。我想颁给 OpenAI。虽然今年它负面新闻多，融资也多，**Sam Altman** 到处搞合纵连横，绑定英伟达算力，跟微软、甲骨文周旋。但我认为 OpenAI 厉害的地方在于，它已经脱离了靠强人管理的阶段。今年发行的 **Sora**、**Atlast** 浏览器和 GPT-5，亮出来的产品洞察力都极强。

**老修**: 我补充一下，OpenAI 在编码产品上进步飞快。**Codex** 专为编码做的大模型和 IDE，用户量增加了 20 多倍，价格也很有竞争力。它的产品迭代能力确实很强。但我个人的提名是 **Google AI 团队**。Google 在 GPT 出来后进行了重构，由 **Demis Hassabis** 接手，把 DeepMind、Gemini 团队整合在一起。Google 实现了软硬件及科研三位一体的爆发，是一个“六边形战士”。它从 Gemini 2 开始就用自研的 **TPU** 训练模型，成本比别人低一个数量级。这种数千人级别的神级配合，在大公司里极难实现。

**闲哥**: 我想提一个国内团队：**阿里**。上半年的 DeepSeek 很惊艳，但下半年阿里在 **通义千问** 和 **夸克** 上的表现非常出色。作为一家有电商基因的公司，阿里迅速把 AI 能力落地到 ToC 端，它的 APP 下载量和营收都非常火爆。阿里真正做到了“AI+”，把 AI 加到原有业务里，给全球 AI 公司做了一个很好的场景落地示范。

**小苏**: 阿里的整合确实不容易。它最有意义的一点是开启了“AI 怎么盈利”的教科书打法。如果千问能整合高德、饿了么、飞猪，变成一个超级入口，那它就能跟苹果这样的 OS 谈系统级 Agent 合作。未来 AI 商业模式的出口，阿里给出了很好的示范。

**老修**: 苹果的 **Apple Intelligence** 在中国跳票了，说不定以后合作伙伴就是阿里。

**小苏**: 最终年度最佳团队，我们颁给——**OpenAI**！阿里和 Google 是“躺在金矿上思考”，而 OpenAI 是“平地起高楼”，从几百人膨胀到几千人，依然保持最快的交付能力和敢为天下先的勇气。

### 年度最佳 Boss：从科研工程双通的闫俊杰到身段柔软的奥特曼

**小苏**: 第三项是 **年度最佳 Boss 奖**。我们有三个提名：Hassabis、**闫俊杰** 和 Sam Altman。老修，你为什么提名 **MiniMax** 的创始人闫俊杰？

**老修**: 闫俊杰在科研、工程和消费端全部跑通了。他在海外市场通过音频、视频产品实现了一个亿美金以上的营收，公司估值达到几十亿美金。他本人很年轻，清华博士后背景，从商汤实习生做到副总裁。他早在 2021 年就看到了大模型的机会，在小团队、资源有限的情况下，做出的音频视频模型在全球处于顶尖水平。他非常像 OpenAI 的早期风格，专注且高效。

**闲哥**: 我的提名是 **Sam Altman**。他身段极其柔软，经历过被逼宫又迅速重返，获得了 90% 以上员工的签名支持。他是个顶级的社交花蝴蝶，能把竞争对手的钱搞在一起。更有魄力的是，他任命了一个 23 岁没上过大学的瑞典小哥担任首席 AI 科学家，这种识人能力（伯乐）极其难得。

**小苏**: 最终，年度最佳 Boss 奖颁给——**Demis Hassabis**！虽然奥特曼很有魅力，但 Hassabis 在 Google 这种大象内部成功让大象起舞，这种“孤臣”式的担当和整合能力，值得这份尊重。

### 年度最佳逆袭：甲骨文的老树开花与 Anthropic 的十倍增长

**闲哥**: 下一个奖项是 **年度最佳逆袭奖**。我提名 **Oracle（甲骨文）**。它曾经是 ToB 时代的基建王者，在 AI 时代本该被淘汰。但它今年下半年通过与 Nvidia、OpenAI 的合作，把自己变成了 AI 基础建筑提供商，市值一天暴涨 40%，创始人身家一度冲到全球首富。这种老树开花的“过山车式”转型，非常精彩。

**小苏**: 我提名 **通义千问**。它更名后二十几天内月活做到 3000 万，展示了阿里重塑生态的决心。但我更同意老修的提名——**Anthropic**。

**老修**: 没错，逆袭奖颁给 **Anthropic**。去年这时候，它只是 OpenAI 跑出来的几个人的小公司。但今年，它的估值冲到了上千亿美金。最惊人的是收入：2024 年底收入约 8.5 亿美金，现在年化运行率（Run Rate）已经达到了 90 亿美金，一年实现 10 倍增长！它通过 Claude Code 赚到了真钱，把服务企业数量从不到 1000 家干到了 30 万家。它是 AI 时代真正的原生公司，交付质量极高。

### 特别奖：腾讯的韬光养晦、Karpathy 的趋势引导与反重力 IDE

**闲哥**: 特别奖我给 **腾讯**，奖项名叫“最具韬光养晦奖”。虽然下半年它没怎么发力，但它挖来了大神 **姚顺禹** 担任首席 AI 科学家。这个姚班毕业的天才曾预测了 AI 下半场的推理赛道，他的想法在 OpenAI 得到了验证。背靠腾讯的财力和生态，明年它一定会有大动作。

**老修**: 我颁给 **Andrej Karpathy**，奖项叫“趋势吹哨人”。在迷茫时刻，他总是能温暖地分享。他提出的 **Vibe Coding（氛围编程）** 概念，让很多不编程的人也能通过大模型快速实现想法。他分享的使用技巧，比如如何配合 AI 自动完成代码而不是盲目生成，对我们提高效率非常有帮助。

**小苏**: 我颁给 Google 的新 IDE **Antigravity（反重力）**。它整合了原生 Agent Mode，在授权后能自动帮我清理垃圾、整理文件、翻译字幕。它让我看到了 PC 端 GUI Agent 的巨大机会，很多工具软件以后都不需要了。

### 展望 2026：Agent 普及、AI 硬件与商业化元年

**小苏**: 展望 2026 年，我认为不会再有那么多基础大模型获奖了，国内的创创业者和应用型产品将占据主流。明年将是 **Agent 商业化** 非常关键的一年，我们会看到清晰的盈利模式。

**老修**: 我的期待是 Agent 直接代替我们处理繁琐的小事，比如剪片子、发视频。AI 编程手段的解放就像人类掌握了火，我期待 24 小时工作的数字员工能真正跑起来。

**闲哥**: 我关注两件事：一是 **AI 眼镜** 等新硬件的出现，希望能摆脱手机交互；二是希望中国 AI 企业更“凶”一点，在落地应用和提升幸福感方面拿出更好的产品。

**小苏**: 感谢大家听我们抒发了一个多小时。期待 2026 年，我们一起做大家 AI 信息的分享者。拜拜！
